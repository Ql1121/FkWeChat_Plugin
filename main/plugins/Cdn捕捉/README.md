该插件用于捕获和处理CDN下载和上传的内容，以便进行分析和优化。

## 功能

- CDN下载内容捕获
- CDN上传内容捕获
- 内容分析
- 优化建议
- 多种文件类型支持
- 详细日志记录

## 使用方法

1. 配置插件设置
2. 开始捕获CDN下载和上传内容
3. 分析捕获的内容
4. 应用优化建议
5. 查看详细日志

## 文件类型支持

插件支持以下文件类型：

- 高清图
- 普通图
- 缩略图
- 视频
- 通用文件
- 小视频
- 聊天语音
- 视频号图片
- 朋友圈视频
- 视频号视频
- APP文件

## 日志记录

插件会记录详细的日志信息，包括：

- 下载和上传的文件信息
- 文件类型
- 文件大小
- 文件路径
- 下载和上传的状态

## 优化建议

根据捕获的内容，插件会提供优化建议，以提高CDN的性能和效率。
